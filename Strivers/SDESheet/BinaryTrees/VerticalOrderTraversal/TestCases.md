# Test Cases for Vertical Order Traversal

## Test Case 1

Input:

```
    1
   / \
  2   3
 /     \
4       5
```

Output: [[4], [2], [1], [3], [5]]

## Test Case 2

Input:

```
    3
   / \
  9  20
     /  \
    15   7
```

Output: [[9], [3, 15], [20], [7]]

## Test Case 3

Input:

```
    1
```

Output: [[1]]
